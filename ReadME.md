# ApexFitness

ApexFitness is a responsive fitness website developed using HTML, CSS, JavaScript, and Bootstrap 5. It provides users with information about gym services, membership plans, trainers, class schedules, and allows visitors to submit enquiries through a validated contact form.

## Features

- Responsive design for desktop and mobile devices
- Modern navigation bar with Bootstrap
- Homepage hero section with background image
- Class filtering using JavaScript
- Membership pricing page
- Weekly class schedule
- Trainer profiles
- Fitness blog
- Frequently Asked Questions (FAQ)
- Contact form with client-side validation

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5.3

## Installation

1. Download or clone the repository.

```bash
git clone https://github.com/MarkNyanjui/FitnessApp
```

2. Open the project folder.

3. Open `index.html` in any modern web browser.

No additional installation or dependencies are required since Bootstrap is loaded through a CDN.

## JavaScript Functionality

The project includes two interactive features:

- Class filtering on the Classes page
- Contact form validation with success and error messages

## Responsive Design

Bootstrap's grid system and utility classes were used to ensure the website adapts to different screen sizes.
